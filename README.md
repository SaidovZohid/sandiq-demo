# sandiq-demo

## Setup:
1. Clone project:
```sh
git clone git@github.com:SaidovZohid/sandiq-demo.git
```

2. Change name `.env.example` to `.env` and add your configs

3. Start project:
```sh
make start
```

4. Migrate project:
```sh
make migrate
```

5. Create super user for admin dashboard
```sh
make superadmin
```

6. Open: http://localhost:8000/
